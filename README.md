# UT4-

Act. 4 — Pedido de Restaurante



Descripción breve:
Tarea sobre representar un Pedido de Restaurante utilizando una clase en Java y el Manejo de github en trabajo dual.



Ejecución del programa:



1. Clonar del repositorio mediante el git bash copiando el enlace del repositorio en GitHub (botón verde Code → SSH) y en Git Bash escribe: git "SSH"



2\. Entrar en la carpeta del proyecto cd repositorio



3\. Abrir el proyecto en tu editor



Por ejemplo en IntelliJ IDEA: File → Open y selecciona la carpeta del repositorio.



4\. Buscar la clase principal.



5\. Ejecutar 



En IntelliJ clic derecho sobre la clase → Run 'Main.main()'
